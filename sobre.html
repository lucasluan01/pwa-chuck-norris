Requisitos:

  • Sua aplicação precisa conter pelo menos duas interfaces (telas ou páginas)
  • Sua aplicação precisa armazenar os dados da lista localmente usando a API Web Storage
  • Sua aplicação não deve compartilhar dados entre usuários (ela não vai rodar em um servidor que possa contar com um banco de dados)
  • Sua aplicação deve ser adequada para execução em smartphones (não será uma aplicação para computadores)
